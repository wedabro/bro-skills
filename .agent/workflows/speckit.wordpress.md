# /speckit.wordpress

Sử dụng skill `speckit.wordpress` để xây dựng website WP chuyên nghiệp.

## Các bước thực hiện:

1. **Khởi tạo môi trường**: Kiểm tra Docker WP, thiết lập folder theme.
2. **Thiết lập Architecture**: Định nghĩa Custom Post Types, Taxonomies, ACF fields.
3. **Phát triển UI**: Code file PHP template kết hợp Tailwind CSS.
4. **Data Sync**: Sử dụng WP-CLI để đồng bộ dữ liệu mẫu.
5. **Security Check**: Gating login, cấu hình permissions, và tối ưu performance.
