# /speckit.map

Sử dụng skill `speckit.map` để thấu hiểu dự án.

## Các bước thực hiện:

1. **Quét cấu trúc**: Phân tích cây thư mục và Tech Stack.
2. **Vẽ sơ đồ phụ thuộc**: Phân tích `import/exports` để thấy sự liên kết.
3. **Inventory**: Liệt kê các tích hợp bên thứ ba (APIs, DBs).
4. **Tạo/Cập nhật Docs**: Ghi vào `.agent/codebase/STRUCTURE.md` và `ARCHITECTURE.md`.
