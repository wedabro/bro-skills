# /speckit.roadmap

Sử dụng skill `speckit.roadmap` để điều phối dự án.

## Các bước thực hiện:

1. **Khởi tạo/Cập nhật Roadmap**: Duy trì file `.agent/ROADMAP.md`.
2. **Quản lý Milestone**: Định nghĩa các mốc quan trọng (MVP, Alpha, Beta).
3. **Phân bổ Requirements**: Đảm bảo mỗi yêu cầu trong Spec được gán vào 1 Phase cụ thể.
4. **Theo dõi tiến độ**: Cập nhật trạng thái hoàn thành của Milestone dựa trên các Phase đã xong.
